## Installation & Setup
### Prerequisites
- Java 17+
- Node.js 16+
- MySQL Server
- Git

### Clone the Repository
```sh
 git clone https://github.com/yourusername/supply-tracker-system.git
 cd supply-tracker-system
```

### Backend Setup
```sh
cd backend
mvn clean install
mvn spring-boot:run
```

### Frontend Setup
```sh
cd frontend
npm install
npm start
```
